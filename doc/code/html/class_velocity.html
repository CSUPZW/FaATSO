Detailed Description

the class Velocity corresponds to an anisotropic seismic velocity model, characterised by two independent parameters.

the model is very simple, and is more phenomenological than purely physical. the model is for transversely isotropic medium, and the phase velocity is elliptical: where is the phase angle, is the horizontal wavespeed, and is the anisotropy parameter.
